# Практическая работа №1 - Линейная регрессия

## 1. Цель работы
Изучить применение методов линейной регрессии с использованием библиотеки  

### В результате должна быть создана программа, которая:
- анализирует данные,
- обучает модель линейной регрессии,
- строит графики,
- делает предсказание значения зависимой переменной.

## 2. Структура проекта

```text
project/
│
├── main.py                    # основной скрипт
├── config.py                  # конфигурационный файл
├── requirements.txt           # зависимости проекта
├── data.csv                   # входные данные
├── correlation_matrix.csv    # матрица корреляции (генерируется)
└── README.md                  # инструкция
```


## 3. Требования

2. Название файла: **data.csv**
3. В датасете должны присутствовать **числовые столбцы**



### 4.5. Запуск программы

```bash
python main.py
```

## 5. Что делает программа

1. Загружает данные из `data.csv`
2. Рассчитывает матрицу корреляции признаков
3. Сохраняет матрицу корреляции в файл `correlation_matrix.csv`
4. Строит график корреляционной матрицы (Рис. 1)
5. Автоматически выбирает два наиболее коррелирующих признака
6. Отображает распределение данных (Рис. 2)
7. Делит данные на обучающую и тестовую выборки
8. Обучает модель линейной регрессии
9. Рассчитывает:
    - коэффициент наклона
    - точку пересечения
    - MSE
    - MAE

10. Строит линию линейной регрессии (Рис. 3)
11. Запрашивает значение у пользователя и делает предсказание


## 6. Ввод пользователя

После выполнения расчётов программа запросит:

```text
Введите значение признака <X>:
```

Необходимо ввести **числовое значение**.
После этого будет выведено предсказанное значение зависимой переменной.


## 7. Выходные данные

В результате работы программы:
- выводятся данные и расчёты в консоль
- отображаются 3 графика:
  - матрица корреляции
  - распределение данных
  - линия линейной регрессии
- создаётся файл `correlation\_matrix.csv`


## 8. Возможные ошибки

- `FileNotFoundError: data.csv not found`: Проверьте, что файл `data.csv` лежит в папке проекта
- `ValueError: could not convert string to float`: Вводите \*\*только числовые значения\*\*
- Нет графиков: Проверьте, что установлен `matplotlib`
